https://github.com/atmollohan/atmollohan.github.io

This portfolio is built with Gatsby (React-based static site generator), styled with Sass/SCSS, 
and deployed to GitHub Pages via GitHub Actions. The site showcases my work experience, skills, 
and personal projects.

Tech Stack:
- Framework: Gatsby 5 (React 18)
- Styling: Sass/SCSS with modern @use syntax
- Testing: Jest with React Testing Library
- Linting: ESLint with Prettier
- CI/CD: GitHub Actions (lint, test, build, deploy)
- Hosting: GitHub Pages

How to Run:
1. Install Node.js (v24.8.0) via nvm: nvm install && nvm use
2. Install dependencies: npm install
3. Start dev server: npm run dev
4. Run tests: npm test
5. Lint code: npm run lint
6. Format code: npm run format
7. Build for production: npm run build
8. Deploy: npm run deploy
