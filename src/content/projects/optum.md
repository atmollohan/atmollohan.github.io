## Overview

Optum is the technology subsidiary of United Health Group, the largest health insurance provider in the country, serving over 49 million members.

## Role

Started as Technology Development Intern, then full-time Technology Development Associate, promoted to Software Engineer. Worked on internal enterprise applications supporting healthcare operations.

## Tech Stack

- **Frontend**: React, Next.js
- **Backend**: Node, Java, Python
- **Database**: Cassandra, MySQL, MongoDB, TigerGraph
- **Cloud**: GCP
- **ML**: TensorFlow, PyTorch
- **Enterprise**: Mainframe integration, legacy systems

## Key Projects

### Web Application Development

Built and maintained multiple web applications using React, Next.js, and Node.js. Created responsive UIs and RESTful APIs for enterprise healthcare systems.

### Claims Processing System

Worked on internal systems for processing insurance claims. Built APIs and web interfaces for claims intake and processing using Java and SQL.

### Member Portal

Contributed to customer-facing portal for members to view claims, coverage, and benefits information using React and Node.

### Data Integration

Built integration pipelines connecting various internal systems. Worked with large datasets and batch processing using Cassandra, MongoDB, and TigerGraph.

- Received return offer during internship (junior year)
- Promoted from intern to full-time to associate to engineer
- Gained foundational enterprise software experience
- Learned healthcare industry domain knowledge
