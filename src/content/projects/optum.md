## Overview

Optum is the technology subsidiary of United Health Group, the largest health insurance provider in the country, serving over 49 million members.

## Role

Started as Technology Development Intern, then full-time Technology Development Associate, promoted to Software Engineer. Worked on internal enterprise applications supporting healthcare operations.

## Tech Stack

- **Backend**: Java, SQL
- **Database**: Oracle
- **Frontend**: Angular
- **Enterprise**: Mainframe integration, legacy systems

## Key Projects

### Claims Processing System

Worked on internal systems for processing insurance claims. Built APIs and web interfaces for claims intake and processing.

### Member Portal

Contributed to customer-facing portal for members to view claims, coverage, and benefits information.

### Data Integration

Built integration pipelines connecting various internal systems. Worked with large datasets and batch processing.
