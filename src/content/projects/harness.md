## Overview

Harness is a leading cloud software development platform. After ZeroNorth was acquired by Harness in 2021, I helped build the first security center product, Security Testing Orchestration (STO).

## Role

Started as Software Engineer II, promoted to Senior Software Engineer I, then Senior Software Engineer II. Contributed across the full stack while specializing in platform integrations and security tooling.

## Tech Stack

- **Backend**: Go, Python
- **Security Tools**: Integrated with 20+ security scanners (SAST, DAST, SCA, container scanning)
- **Cloud**: GCP, Kubernetes, Docker, Terraform
- **Frontend**: React, TypeScript, Webpack, Microfrontends

## Key Projects

### STO Core Platform

Built the core orchestration engine that automates security scanning across the software delivery pipeline. Enabled teams to run security tests as part of CI/CD without manual intervention.

### Scanner Integrations

Implemented integrations with 20+ security scanning tools including Snyk, SonarQube, Checkmarx, Black Duck, Qualys, and others. Created standardized interfaces for running scans and normalizing results.

### Internal Code Management Migration

Led migration of internal code management from third-party platform to Harness's own Git-based solution. Coordinated a complex cross-team effort with minimal disruption.

### Platform Extensibility

Built extensibility features allowing customers to create custom scanner integrations and workflows. Designed API surface for third-party integrations.
