## Overview

ZeroNorth was a Boston-based startup providing a platform for automating and orchestrating security scanning across enterprise environments.

## Role

As a Software Engineer II, I worked as a full-stack developer building both the frontend interface and backend services for the security platform.

## Tech Stack

- **Frontend**: Angular, TypeScript
- **Backend**: Node.js, TypeScript libraries
- **Security**: Python-based security runners, integration with 15+ scanners
- **Cloud**: AWS

## Key Projects

### Security Orchestration Engine

Built the core engine that orchestrates security scans across multiple tools and environments. Designed workflows for scheduling, parallel execution, and result aggregation.

### Scanner Integrations

Integrated with 15+ security scanning tools including SAST, DAST, and SCA tools. Created standardized interfaces for running scans and normalizing findings.

### User Interface

Developed Angular-based user interface for configuring scan schedules, viewing results, and managing remediation workflows. Created intuitive dashboards for security teams.

### API Development

Built RESTful APIs for scanner management, scan execution, and result retrieval. Implemented authentication and authorization patterns.

## Impact

- Helped establish core security orchestration capabilities
- Gained deep experience in cybersecurity domain
- Developed strong full-stack skills working across the entire stack
