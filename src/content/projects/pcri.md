## Overview

PCRI (Professional Conference and Research Institute) hosts an annual summit for healthcare professionals. I served as event technology consultant for two consecutive years, managing the technical aspects of their virtual and hybrid conferences.

## Role

- Managed event registration and ticketing through Eventbrite
- Configured and managed Zoom for virtual/hybrid event delivery
- Built integrations between Eventbrite and Zoom for seamless attendee experience
- Created scripts to generate attendee credential lists from both platforms
- Ensured reliable event delivery for conferences with 500+ attendees

## Tech Stack

- **Event Management**: Eventbrite
- **Virtual Events**: Zoom
- **Integration**: Eventbrite API, Zoom API
- **Automation**: Python scripting

## Key Projects

### Event Integration

Built automated integration between Eventbrite and Zoom to sync registration data, generate unique join links, and manage attendee access.

### Credential Management

Developed Python scripts to extract and merge attendee data from Eventbrite and Zoom APIs, generating credential lists for event access and check-in.

### Event Execution

Managed live event technical requirements including breakout rooms, virtual networking sessions, and Q&A infrastructure.

## Impact

- Successfully executed two annual summits with 500+ attendees each
- Streamlined registration process through API automation
- Created repeatable processes for future PCRI events
