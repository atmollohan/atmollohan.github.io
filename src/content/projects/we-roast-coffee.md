## Overview

We Roast Coffee is a local coffee company in northern Rhode Island. I worked with the founder to build the digital presence and online store.

## Role

Co-founded technical side of the business, building and maintaining the e-commerce platform and digital assets.

## Tech Stack

- **Frontend**: Gatsby (React), JavaScript
- **E-commerce**: Shopify
- **CMS**: Contentful
- **Auth**: Auth0
- **Hosting**: Netlify

## Key Projects

### Shopify Store

Built online store with Shopify, hosted on a memorable domain (weroast.coffee). Integrated payment processing, inventory management, and shipping calculations.

### JAMstack Blog

Initially built a proof-of-concept blog using JAMstack (JavaScript, APIs, Markdown) with Gatsby. Later replaced with the Shopify store.

### Authentication

Implemented user authentication and authorization using Auth0 for customer accounts and admin access.

### Custom Domain

Configured memorable domain and DNS settings for seamless customer experience.
